# Space-Battle

## Goal

Shoot down the enemy spaceship. You each get 10 lives, whoever defeats the opponent wins! 

## Controls

Use the arrows to move the red spaceship or WASD to move the yellow spaceship. Shoot the bullets with ctrl keys.
